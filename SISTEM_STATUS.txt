✅ SISTEM TUGAS - STATUS KONEKSI

=== ALUR KERJA ===

1. ADMIN MEMBUAT TUGAS
   - URL: http://localhost/coba/admin/tasks.php
   - Admin mengisi: Judul, Tanggal, Catatan
   - Admin memilih pengguna yang akan menerima tugas
   - Tugas disimpan dengan status dan progress

2. TUGAS OTOMATIS MUNCUL KE USER
   - User Biasa login: http://localhost/coba/auth/login.php
   - Dashboard User: http://localhost/coba/user/user_dashboard.php
   - Semua tugas dari admin muncul di sini
   - Tersortir: Hari ini, Mendatang, Selesai

3. USER DAPAT MELIHAT DETAIL & SUBTASK
   - Klik task card → http://localhost/coba/user/user_task.php?id=X
   - User bisa tandai subtask selesai
   - Progress task otomatis terupdate

4. HALAMAN HUB TERPUSAT
   - URL: http://localhost/coba/user/user_task_detail.php
   - Menampilkan statistik keseluruhan
   - Link ke semua fitur user dan admin
   - Akses cepat ke:
     * Dashboard Tugas
     * Admin Hub
     * Buat Tugas Baru (Admin)
     * Kelola Users (Admin)
     * Laporan (Admin)

=== DATABASE SCHEMA ===

Tabel: tasks
- id (int, auto increment)
- title (varchar) - Nama tugas
- status (varchar) - todo, progress, completed
- progress (int) - 0-100%
- start_date (varchar)
- end_date (varchar) - Deadline
- note (text) - Deskripsi
- assigned_users (text) - Nama user, dipisah koma
- created_by (varchar) - Username admin pembuat
- tasks_completed (int) - Subtask yang selesai
- tasks_total (int) - Total subtask
- created_at (timestamp)

Tabel: task_subtasks
- id, task_id, title/subtask_name, assigned_to, is_completed
